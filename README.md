# node-typescript-seed

Typescript seed for Node.js projects

## Usage

- Setup
  1. `yarn install`

- Compile Code
  1. `npm run tsc`
  2. `npm run tsc:watch`

- Testing
  1. `npm test`
  2. `npm test:watch`

- Code Coverage
  1. `npm run coverage`
  2. `open coverage/index.html`

- Linting
  1. `npm lint`
  2. `npm lint:watch`
